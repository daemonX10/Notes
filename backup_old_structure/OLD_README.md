# Organized Interview Questions

## Question Types

Each topic folder contains questions categorized by type:

- **Theory**: Conceptual questions about the topic
- **Coding**: Programming challenges and implementation questions
- **Scenario Based**: Real-world application and problem-solving questions
- **General**: Other types of questions
